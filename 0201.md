## 0201 
# 만약 빛이 없으면 ( if~else 조건문 )


만약 로또에 당첨된다면?

만약 남자(여자)로 태어난다면?

어떤 상황을 전제하느냐는 이후의 많은 것을 달라지게 합니다. 이를 프로그래밍에서는 if문으로 제어합니다.  
```
if(로또에 당첨된다면){
 console.log(건물살래);
}
```
실제로 돌아가는 코드를 볼까요.
```
// src-02-01

let letThereBeLight = true; // or false
if(letThereBeLight){  
  console.log('안녕세상');
} else {  
  console.log('암흑암흑');
}
```
위 코드를 *'세상을 만드는 코드'*라고 부르겠습니다. 
이번 챕터가 끝날 무렵 우리는 코드로 창세기를 쓰게 됩니다.(설마)
